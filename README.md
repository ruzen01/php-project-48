# Gendiff

## Usage

### JSON Format

```sh
$ ./bin/gendiff --format json ./tests/fixtures/file1.json ./tests/fixtures/file2.json
{
    "common": {
        "setting1": "Value 1",
        "setting2": "200",
        "setting3": true
    },
    "group1": {
        "baz": "bas",
        "foo": "bar"
    }
}
```
